Troubleshooting I see message about "JetBrains Chrome Extension for Debugging and Live Editing will be automatically installed after you restart Google Chrome", but extension is not installed after Chrome restart IDE tries to register JB extension in Google Chrome (and Google Chrome Canary). So, possible reasons: You use Chromium-based browser or you use Linux (chrome bug). You can manually add extension. Chrome is not active. Ensure that Settings -> Web Browsers -> Chrome marked as Active. How to enable logging in Google Chrome Mac OS X: open -a "Google Chrome" --args --enable-logging --v1 How do I know what's wrong? Open chrome://chrome/extensions/ Check "Developer mode" Inspect views: _generated_background_page.html (JetBrains Chrome Extension) Open "Console" Tab Or enable Google Chrome logging. Jetbrains Extension is not registered for Google Chrome under Linux (Ubuntu).
